Alem's Lua Executor (Lua v5.3.4)

Alem's Lua Executor (Lua v5.3.4) helps you to Edit, Run and Share your Lua Code directly from your browser. This development environment provides you version Lua v5.3.4.

How to give program Input?

The latest version of Alem's Lua Executor allows to provide program input at run time from the termnial window exactly the same way as you run your program at your own computer. So simply run a program and provide your program input (if any) from the terminal window available in the right side.
